1. 라이브러리 설치
    - pom.xml의 dependencies 안에 mysql, mybatis 라이브러리 추가
        <dependency>
            <groupId>com.mysql</groupId>
            <artifactId>mysql-connector-j</artifactId>
        </dependency> (MySQL)

        <dependency>
            <groupId>org.mybatis.spring.boot</groupId>
            <artifactId>mybatis-spring-boot-starter</artifactId>
            <version>3.0.5</version>
        </dependency> (MyBatis)

    - maven update

2. application.properties 또는 application.yml 세팅
    - spring.datasource 세부 세팅 (들여쓰기는 두칸)
        driver, url, username, password
    ex)
    spring:
      application:
        name: springboot
      datasource:
        driver-class-name: com.mysql.cj.jdbc.Driver
        url: jdbc:mysql://localhost:3309/sb_study_yjw
        username: root
        password: 1q2w3e4r

    springdoc:
      swagger-ui:
        path: doc

    - mybatis 세부 세팅
    ex)
    mybatis:
      mapper-locations: classpath:mapper/**/*.xml -> Mapper.xml 파일을 찾을 수 있도록 경로 지정
      configuration:
        map-underscore-to-camel-case: true -> 카멜표기법 <-> 스네이크표기법 서로 변환할수 있게 설정
      type-aliases-package: com.korit.springboot.entity -> Mapper.xml에서 entity 객체 클래스 경로지정

3. mybatis Mapper 패키지 또는 xml 폴더 생성
    - 패키지는 Controller, Service, Dto, Entity 등과 같은 레벨에 생성
    - xml 폴더는 resources 폴더 하위에 mapper 생성
    - mapper.xml 파일 안에 꼭 들어가야하는 mapper 설정 태그 라이브 템플릿 세팅 (mybatis 공식 홈페이지에서 참조)
    ex)
    <?xml version="1.0" encoding="UTF-8" ?>
    <!DOCTYPE mapper
            PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
            "https://mybatis.org/dtd/mybatis-3-mapper.dtd">
    - 라이브 템블릿 세팅은 Ctrl + Alt + S 눌러서 설정을 들어간 후 라이브 템플릿 검색
      좌측상단 +를 눌러서 약어와 설명은 ex) st, Exploring Mapped SQL Statements로 설정
      템플릿 텍스트에
        <?xml version="1.0" encoding="UTF-8" ?>
            <!DOCTYPE mapper
                    PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
                    "https://mybatis.org/dtd/mybatis-3-mapper.dtd"> 내용을 넣어준 후 XML로 잡아준다.
